# How to contribute ?

This repository is open to new features, bug fixes, security advisories... Please, open a **well documented** issue
to disclose your needs and then, eventually open a PR.


## How to launch the project ?

Just run : 

```bash
cargo run
```

Please, write unit tests proving that your changes aren't breaking the project.
To run the tests, just run : 

```bash
cargo test
```

This project uses pre-commit, to set-up pre commits on that repository run : 

```bash
pre-commit install
```
